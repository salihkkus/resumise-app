# Resumise 🚀
**NLP Tabanlı Akıllı CV Analiz ve Kariyer Sistemi**

Resumise, adayların teknik yetkinlikleri ile sektör beklentileri arasındaki boşluğu (skill gap) tespit eden yapay zeka destekli bir platformdur. 

## 🛠️ Mimari (Architecture)
- **Frontend:** React.js 
- **Backend:** Java Spring Boot 3.x 
- **AI Engine:** Python FastAPI & Sentence-Transformers (BERT) 
- **LLM:** Google Gemini API
